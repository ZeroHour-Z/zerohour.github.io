# 绪论

![image-20260320012723319](image-20260320012723319.png)

## 1.1 运动控制系统及其组成

- 控制器——模拟/数字 mcu
- 功率放大器与变换装置——电力电子设备
- 电动机及负载——被控对象
- 传感器
- 信号处理

运动控制系统概念：以机械运动的驱动设备——**电动机**为被控对象，以**控制器**为核心，以电力电子功率变换装置为**执行机构**，在自动控制理论指导下组成的自动控制系统

运动控制系统功能：主要控制电动机的**转矩**、转速和转角，将电能转化为机械能，实现运动机械的运动要求

## 1.2 运动控制系统分类

按被控物理量分：

- 调速系统：以转速为被控量
- 伺服系统：以位移（角位移或直线位移）为被控量

按驱动电机分：

- 直流控制系统：直流电机带动
- 交流控制系统：交流电机带动

按控制器类型分：

- 模拟控制系统：控制器是模拟电路
- 数字控制系统：控制器是数字电路

按闭环数分：

- 单环系统
- 双环系统
- 多环系统

## 1.3 运动控制系统的历史与发展

发展趋势：

1. 高频化
2. 网络化
