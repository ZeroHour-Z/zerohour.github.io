# 第 1 章 引论

## 1.1 计算机网络的产生与发展

### 工业互联网
通过一个开放的、全球化的通信网络平台，把设备、生产线、员工、 工厂、仓库、供应商、产品和客户紧密地连接起来，共享工业生产全流程的各种要素资源，使其数字化、网络化、自动化、智能化，从而实现效率提升和成本降低 

### 分为四个时代
+ 以单计算机为中心的多终端联机系统：共享主机资源
+ 以通信子网为中心的计算机—计算机网络
+ 采用标准化层次体系结构的网络
+ Internet时代

### 通信子网
由CCP组成的传输网络，面向通信控制盒通信处理

### 资源子网
主机的集合，拥有资源的用户主机和请求资源的用户终端

### 计算机网络
计算机网络是将地理位置不同、具有独立功能的多台计算机及其外部设备通过通信线路和设备连接起来，实现资源共享和信息传递的系统</font>

利用各种通信手段，在网络协议控制下，把地理上分散的自治计算机有机地连在一 起，达到互相通信且共享软件、硬件和数据等资源的系统</font>

### 计算机网络的四大基本特征
+  计算机网络建立的主要目的是实现计算机资源的共享  
+ 互联的计算机是分布在不同地理位置的多台独立的自治计算机
+ 连网计算机必须遵循全网统一的网络协议
+ 通信网是计算机网路的一个基本要素

### 计算机网络的组成

1. 软件
    1. 操作系统：软硬件设备的控制程序
    2. 协议和通信软件：控制网络中的设备之间如何进行信息交换
    3. 管理及应用软件：对网络资源进行管理维护的，为用户提供服务
2. 硬件
    1. 层次：由通信子网和资源子网组成
    2. 类型（组成元素）
        1. 网络节点：端节点（主机，终端）和转接节点（交换机、路由器、HUB）
        2. 通信链路：传输信息的信道

### 计算机网路的分类
1. 按距离
    1. 局域网：范围小；基带传输，延迟低，出错率低；总线型，星形，树形，环形
    2. 广域网：范围大；频带传输，延迟大，出错率高；不规则，点到点
2. 按通信介质
    1. 有线网：同轴电缆，双绞线，光纤
    2. 无线网：卫星，微波
3. 按使用范围
    1. 公用网
    2. 专用网
4. 按拓扑结构
    1. 星形
    2. 环形，首尾相连的总线形
    3. 树形
    4. 总线
    5. 全连接/点到点
    6. 不规则，多用于广域网
