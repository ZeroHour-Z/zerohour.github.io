## 🚀 本地开发

### 环境要求

- Node.js >= 16
- pnpm

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm run dev
```

访问 `http://localhost:4860/` 查看效果。

### 构建

```bash
pnpm run build
```

构建产物在 `dist/` 目录。

## 📁 项目结构

```
├── pages/           # 页面文件
│   ├── posts/       # 博客文章
│   ├── about.md     # 关于页面
│   └── index.md     # 主页
├── components/      # 自定义组件
│   ├── AnimatedCard.vue    # 动画卡片组件
│   └── TypewriterText.vue  # 打字机效果组件
├── styles/          # 样式文件
│   └── index.scss   # 全局样式
├── snippets/        # 代码片段
├── public/          # 静态资源
├── valaxy.config.ts # Valaxy 配置
└── site.config.ts   # 站点配置
```
